 ### 3. Tidak Ada Hash Verification

  Size check saja tidak cukup untuk validasi. Tambahkan SHA256
 verification:

 ```typescript
     // Contoh untuk model ggml-small.bin di huggingface
     // Bisa tambahkan hash di AVAILABLE_MODELS
     const crypto = require('crypto');

     async verifyHash(filePath: string, expectedHash: string):
   Promise<boolean> {
       return new Promise((resolve) => {
         const hash = crypto.createHash('sha256');
         const stream = fs.createReadStream(filePath);
         stream.on('data', (data) => hash.update(data));
         stream.on('end', () => resolve(hash.digest('hex') ===
   expectedHash));
         stream.on('error', () => resolve(false));
       });
     }
 ```
