 ### 2. Disk Space Check ✅ (DITAMBAHKAN)
  - Method `checkDiskSpace()` menggunakan `fs.promises.statfs` (Node.js native)
  - Cek sebelum download di kedua file:
    - `cudaDownloader.ts` → di `download()`
    - `modelDownloader.ts` → di `downloadModel()` dan `resumeDownload()`
  - Required space = sizeBytes × 1.1 (10% buffer)
  - Jika tidak cukup, return error message dengan detail (need vs available)

 ### 3. Hash Verification ✅ (DITAMBAHKAN)
  - Method `verifySha256()` menggunakan `crypto.createHash('sha256')` dengan streaming
  - Field `sha256?: string` ditambahkan ke interface `ModelInfo`
  - AVAILABLE_MODELS: semua model punya field `sha256` (di-set `undefined` untuk diisi nanti)
  - Constanta `CUDA_ZIP_EXPECTED_SHA256` ditambahkan di cudaDownloader.ts
  - Jika hash tidak di-set, skip dengan log warning
  - Jika hash mismatch, file dihapus dan return error

 ### 4. Retry Logic ✅ (DITAMBAHKAN)
  - Method `downloadWithRetry()` dengan exponential backoff (1s, 3s, 9s)
  - Max 3 retries (total 4 attempts)
  - Hanya retry untuk transient errors:
    - Network errors (connection reset, timeout)
    - HTTP 408 (Request Timeout)
    - HTTP 429 (Rate Limit)
    - HTTP 520+ (Server errors)
  - Tidak retry untuk 4xx/5xx non-transient
  - UI update: kirim retry status ke renderer
  - Request timeout 30 detik ditambahkan di option

 ### 5. Resume Logic Bug ✅ (SUDAH FIX)

  - Saat server return 200 (no resume support), temp file dihapus dan restart dari 0
  - totalBytes di-recalculate dari content-length response baru

 ### 6. Taskbar Progress API ✅ (SUDAH FIX)
